Hardware support layers

In addition to network-related functionality BitCloud provides special set of APIs supporting common hardware interfaces.

  • Hardware abstraction layer is responsible for all interactions between Atmel modules and external periphery.
  • Border support package includes a complete set of drivers for managing standard peripherals (such as buttons, sensors etc.) placed on the development board.
  • Persistent data storage includes functions for reading and writing EEPROM protected from accidentally overwritting stack paramteres.
