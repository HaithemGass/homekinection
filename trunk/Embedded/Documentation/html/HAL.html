Hardware Abstraction Layer

HAL component of the BitCloud SDK provides functions for working with common hardware interfaces such as USART, TWI, SPI, ADC, GPIO, IRQ. Other interfaces supported by HAL include Application timer, System time, Watchdog, Warm Reset

Although available in the HAL component, HAL_ReadEeprom/HAL_WriteEeprom and Sleep Timer functions are intended for internal execution inside the stack and applications shall avoid using them.
