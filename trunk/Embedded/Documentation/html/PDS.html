Persistence Data Server

Persistence Data Server or PDS is a special API responsible for managing, reading and writing non-volatile memory storage available on most microcontrollers which is called EEPROM.

Although HAL component has functions for reading and writing EEPROM, it is strongly recommended to use PDS component for such purposes, because HAL functions access memory directly at given address and hence there is a risk of overwriting internal stack-specific variables. At the same time, PDS uses a special offset to write and read all application data and this protects EEPROM portion occupied by stack parameters from being written over by the application code
